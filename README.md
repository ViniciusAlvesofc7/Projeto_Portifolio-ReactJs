# Portifolio Pessoal

  Projeto de portfolio pessoal desenvolvido para apresentar meus projetos, 
habilidades e formas de contato.

# Tecnologias

- HTML5
- CSS3
- JavaScript
- ReactJS
- Vite
- TailwindCSS
- GSAP

# Funcionalidades

- Layout responsivo
- Animações com GSAP
- Download de currículo em PDF
- Navegação suave entre seções
- Modal menu para dispositivos móveis

## Como Rodar

```bash
# Clone o repositório
git clone https://github.com/ViniciusAlvesofc7/Projeto_Portifolio-ReactJs

# Acesse a pasta
cd portifolio

# Instale as dependências
npm install

# Rode o projeto
npm run dev
